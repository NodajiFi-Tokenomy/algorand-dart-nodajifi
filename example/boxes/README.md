# PyTEAL

## Install Python Virtual Environment

```
python3 -m venv venv
```

## Activate Virtual Environment

```
. venv/bin/activate
```

## Visual Studio Code

### Select interpreter

```
Open Command Palette
> Python: Select interpreter -> venv
```

## Install packages

```
pip3 install -r ./requirements.txt
```

## Compile pyTeal

```
. venv/bin/activate
python3 {file}.py
```

## Update PyTEAL

```
pip3 install pyteal --upgrade
```
